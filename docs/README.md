# 프로젝트 소개

해당 프로젝트는 한 주의 점심 메뉴를 추천해주는 서비스입니다.
코치의 이름과 각각 먹지 못하는 음식을 입력해주시면 그에 따른 월~금의 메뉴를 추천해드립니다.


# 기능 
- [X] 초기값 저장하기
- [X] 코치 이름 입력 받기
- [X] 코치 이름 저장하기
- [X] 코치별로 먹지 못하는 메뉴 입력 받기
- 각 요일별로
  - [X] 추천 카테고리 뽑기
  - 각 코치별로
    - [ ] 메뉴 추천하기
- [ ] 결과 출력하기


# 예외 상황
- [X] 코치 이름이 2글자 ~4글자가 아닌 경우
- [X] 코치가 2명 ~ 5명이 아닌 경우
- [X] 코치가 못 먹는 음식이 0~2개가 아닌 경우

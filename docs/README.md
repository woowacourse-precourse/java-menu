## 기능 목록

## 순서

- [출력] 점심 메뉴 추천 안내
- [출력] 코치의 이름을 입력해 주세요. (, 로 구분)
    - [입력] 코치의 이름 입력 받기
        - [예외] 2분 미만 5분 초과 입력
        - [예외] 2글자 미만 4글자 초과 이름
- [출력] 코치 이름(이)가 못 먹는 메뉴를 입력해 주세요.
    - [입력] 못 먹는 메뉴
    - 메뉴 추천 서비스의 메뉴에 해당하지 않는 경우 == 입력하지 않음
- 메뉴 추천
    - [출력] "[", "|", "]"로 구분
- [출력] 추천 완료 문구

### 도메인

- 메뉴 추천 서비스
    - 카테고리: 메뉴
- 코치
    - [예외] 2글자 미만 4글자 초과 이름
    - [예외] 2분 미만 5분 초과 입력
    - 각 요일의 추천 메뉴
- 카테고리 선정
    - Randoms.pickNumberInRange() 사용
    - [예외] 한 주에 같은 카테고리 2회 초과
- 메뉴 추천
    - Randoms.shuffle() 사용
    - [예외] 한 주에 중복되는 메뉴 추천
    
- 요일
    - 월(0), 화(1), 수(2), 목(3), 금(4)

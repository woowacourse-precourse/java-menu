## 핵심 기능
- [ ] 카테고리를 무작위로 정한다(5번. 월화수목금)
- [ ] 코치마다 못먹는 메뉴를 받는다.
- [ ] 한 주의 메뉴를 추천한다.



## 입출력
- [ ] 메뉴 추천 서비스 시작 문구 출력한다.
- [ ] 코치의 이름을 입력받는다.(,로 구분)
- [ ] 각 코치마다 못 먹는 메뉴가 무엇인지 묻는다.
- [ ] 각 코치마다 못 먹는 메뉴를 입력받는다.
- [ ] 메뉴 추천 결과 문구 출력한다.
- [ ] 메뉴 추천 결과 출력한다.
- [ ] [ERROR]로 시작하는 문구를 출력한다.

## 예외처리
- [ ] 코치의 이름은 2글자 ~ 4글자이다.
- [ ] 코치는 최소 2명, 최대 5명이다.
- [ ] 한 주에 같은 카테고리는 2회까지만 허용이다.
- [ ] 한 주의 메뉴는 중복되지 않는다.

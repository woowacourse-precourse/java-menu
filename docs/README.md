### 기능 요구 사항
1. 서비스 시작 및 입력 받은 값 초기화
- [ ] 서비스 시작 문구 출력 
- [ ] 메뉴 추천을 받을 코치의 이름 입력받는 기능
  - [ ] 코치의 이름을 입력해주세요 
  - [ ] 올바른 값이 아니면 예외 처리 
    - [ ] 최소 2글자, 최대 4글자 
    - [ ] 최소 2명, 최대 5명
- [ ] 각 코치가 못 먹는 메뉴 입력받는 기능
  - [ ] 최소 0개, 최대 2개 
  - [ ] 먹지 못하는 메뉴가 없으면 빈 값 
- [ ] 잘못된 값 입력시 에러 메세지 출력 후 그 부분부터 다시 입력

2. 메뉴 추천 서비스 실행
- [ ] 카테고리 랜덤 추출 기능
  - [ ] 한 주에 같은 카테고리 최대 2회 -> 체크 기능
- [ ] 메뉴 랜덤 추출 기능
  - [ ] 중복되는 메뉴 체크 기능
  - [ ] 먹지 못하는 메뉴 체크 기능

3. 서비스 종료
- [ ] 결과 출력 기능

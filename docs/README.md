## 🍳구현할 기능 목록

### 1. 입력 받는 기능

- [x] 코치의 이름 입력받기 (,로 구분)
- [x] 각 코치별 못먹는 음식 입력 받기

### 2. 출력 받는 기능

- [x] 구분(요일) 출력
- [x] 카테고리 출력
- [x] 각 카테고리는 일주일에 최대 2개까지 중복가능
- [x] 각 코치에게 추천하는 메뉴는 중복 불가

### 3. 예외처리

- [x] 코치의 이름은 2 ~ 4글자
- [x] 코치는 2명 ~ 5명
- [x] 각 코치는 못먹는 메뉴가 0 ~ 2개(,로 구분해서 입력)
- [x] 한 주의 카테고리는 최대 2개까지 허용
- [x] 코치에게 중복되는 메뉴 x

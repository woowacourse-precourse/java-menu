## 기능 목록
### 입출력
#### 입력
- [x] 코치의 이름 입력
- [x] 코치가 못먹는 음식 입력
#### 출력
- [x] 시작 메시지 출력
- [x] 코치 이름 입력 메시지 출력
- [x] 코치가 못먹는 음식 입력 메시지 출력
- [x] 메뉴 추천 결과 출력
### 도메인
- [x] 음식 카테고리를 정의한다.
- [x] 요일을 정의한다.
- [x] 요일별로 카테고리를 무작위로 정한다.
  - [x] 한 주에 같은 카테고리는 최대 2번
- [x] 코치에게 요일별 카테고리에 해당하는 음식을 1개씩 추천한다.
  - [x] 한 코치가 같은 음식을 먹으면 안된다.
- [x] 코치의 이름과 못먹는 음식을 정한다.
## 예외 사항
- [x] 코치의 이름이 최소 2글자 최대 4글자가 아닌 경우
- [x] 코치가 2명 ~ 5명이 아닌 경우
- [x] 코치가 못 먹는 메뉴가 2개보다 많을 경우
